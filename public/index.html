🎥 MovieTime API Documentation

Base URL: https://movie-time-api.onrender.com/api

🔐 Authentication & Security

Operations marked with API KEY require the header: x-api-key: [YOUR_PRIVATE_KEY]

🟢 GET - Retrieval Requests

GET /health

System Status

{ "status": "UP", "moviesCount": 60, "serverTime": "2026-03-24T21:49:12Z" }
GET /movies

Get All Movies

Fetches the complete library of movies.

GET /movies?filters

Movie Filtering & Search

Query ParameterTypeUsage Example
idInteger/movies?id=25
qString/movies?q=Gladiator
genreString/movies?genre=Action
yearInteger/movies?year=2024

🟡 POST - Create Requests

POST /register

User Registration

Register new account. Mandatory: name, email, password, password_confirmation.

Request Body:

{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "password123",
  "password_confirmation": "password123"
}

POST /movies API KEY

Create New Movie

{ "id": 12345, "title": "Gladiator II", "genre": "Action", "duration": 148 }
POST /orders

Book Tickets

Process ticket reservation. Quantity limit: Max 8.

Request Body:

{
  "userId": 1,
  "movieId": 25,
  "quantity": 5
}
