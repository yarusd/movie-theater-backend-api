ברוכים הבאים לפלטפורמת הנתונים וההזמנות הקולנועית (Cinematic Data & Booking Platform) המתקדמת ביותר. המערכת מציעה גישה מהירה לקטלוג סרטים מקיף וכלים מתקדמים לניהול הזמנות מושבים, הכל בחיבור דיגיטלי מאובטח.

Base URL: https://movie-time-api.onrender.com/api
GET /health

System Diagnostics

Verify server availability and database size.

Response (200 OK):

{ "status": "UP", "moviesCount": 60, "serverTime": "2026-03-24T21:00Z" }
GET /movies

Get Full Movie List

Returns the entire movie catalog without any filters.

Response (200 OK):

[ { "id": 1, "title": "ThunderBolts*", "genre": "Action", "rating": 7.6 }, ... ]
GET /movies?filters

Movie Search & Filtering

Search movies using smart query parameters. Partial words and combined filters supported.

ParamTypeExample / Description
qStringSmart search (Title, Genre, Cast). e.g. ?q=Dune
genreStringFilter by genre (Action, Comedy). e.g. ?genre=Action
yearIntegerFilter by release year. e.g. ?year=2024
badgeStringNOW SHOWING, COMING SOON, EXTENDED RUN
sortStringUse sort=rating for highest-to-lowest rating.

Request:

GET /api/movies?q=Gladiator&genre=Action&sort=rating

Response (200 OK):

[ { "id": 25, "title": "Gladiator II", "genre": "Action", "rating": 7.2 }, ... ]
GET /movies/:id/occupied

Occupied Seats Check

Checks which seats are taken for a specific movie session. Query params: date (YYYY-MM-DD), time (HH:mm).

Response (200 OK):

["A1", "C5", "E8"]
GET /orders/:userId

User Order History

Retrieves all bookings for a specific user ID. Requires x-user-id header.

Response (200 OK):

[ { "orderId": "ORD-123", "movieId": 25, "seats": ["A1"], "totalAmount": "₪35.00" } ]
POST /register

User Account Creation

Creates a new user account. Validation rules apply.

Request Body:

{ "name": "Alice Cohen", "email": "alice@test.com", "password": "securepass123", "password_confirmation": "securepass123" }

Response (201 Created):

{ "message": "User registered successfully", "userId": 17112345 }

POST /movies

Add New Movie

Requires Admin role and x-api-key header.

Request Body:

{
  "title": "Avatar 3",
  "genre": "Sci-Fi",
  "duration": 150,
  "year": 2025,
  "badge": "COMING SOON"
}

Response (201 Created):

{ "id": 1711234888, "title": "Avatar 3", ... }
POST /orders

Free Seat Booking

Directly reserve seats without payment details. Max 8 per order.

Request Body:

{
  "userId": 1,
  "movieId": 25,
  "seats": ["B1"],
  "date": "2026-10-10",
  "time": "20:00"
}

Response (201 Created):

{ "orderId": "ORD-1711234567", "status": "confirmed", ... }
POST /payments/checkout

Secure Paid Checkout

Tiered Pricing: Rows A-B (Back: ₪35), C-D (Standard: ₪50), E (Premium: ₪65).

Request Body:

{
  "userId": 1,
  "movieId": 25,
  "seats": ["E1"],
  "cardHolder": "Tzedek Wasi",
  "cardNumber": "1234567812345678",
  "expiry": "12/28",
  "cvv": "123",
  "date": "2026-12-01",
  "time": "19:00"
}

Response (201 Created):

{
  "message": "Payment successful and tickets booked!",
  "order": {
    "totalAmount": "₪65.00",
    "status": "confirmed",
    "transactionId": "TXN-9876543210ABC"
  }
}
PUT /movies/:id

Update Movie

Requires Admin role and x-api-key header.

Request Body:

{ "title": "Updated Title", "genre": "Drama", ... }

Response (200 OK):

{ "message": "Movie updated successfully", "movie": { ... } }
DELETE /movies/:id

Delete Movie

Permanently remove a movie from the catalog. Requires Admin role and x-api-key header.

Response (200 OK):

{ "message": "Movie deleted successfully" }
DELETE /orders/:orderId

Cancel Order

Cancels a booking and frees the seats. Validates ownership.

Response (200 OK):

{ "message": "Order cancelled successfully. Seats are now free." }
DELETE /test/reset

System Reset

Deletes all orders and restores movies and users to the initial factory state. Requires x-api-key header.

Response (200 OK):

{ "message": "System reset to factory state successfully." }

📊 HTTP Status Standards

CodeLabelDescription
200OKRequest completed successfully.
201CreatedResource (Movie, Order, User) created successfully.
400Bad RequestValidation failed. Check English letters, date format, fields.
401UnauthorizedMissing x-api-key header.
402Payment RequiredTransaction declined or Card expired. Check funds.
403ForbiddenAccess denied. Locked account or IDOR attempt.
404Not FoundThe movie or resource does not exist.
409ConflictDuplicate title or seats already booked.
