🎥 MovieTime API Documentation

Base URL: https://movie-time-api.onrender.com/api

🔐 Security & Authorization

This API uses two types of identity verification:

  • Admin Operations: Require x-api-key header for adding, editing, or deleting movies.
  • User Privacy: Personal order history requires x-user-id to prevent IDOR (Insecure Direct Object Reference).

🟢 GET - Data Retrieval

GET /health

System Diagnostics

Returns the status of the server and the current number of movies in the database.

Example Response:

{ "status": "UP", "moviesCount": 60, "serverTime": "2026-03-24T21:00:00Z" }
GET /movies

Movie Catalog & Search

Supports advanced filtering and search queries.

ParameterTypeDescription
qStringSmart search by Title, Genre, or Cast members.
genreStringFilter by category (Case-insensitive).
yearIntegerFilter by release year.
badgeStringFilter by status: NOW SHOWING, COMING SOON, etc.
sortStringUse rating to sort by highest score.
GET /movies/:id/occupied

Occupied Seats Check

Returns an array of strings representing taken seats (e.g., A1, B5) for a specific session.

Query Params: date (YYYY-MM-DD), time (HH:mm)

GET /orders/:userId USER ID

User Order History

Retrieves all confirmed bookings for a specific user ID. Requires matching x-user-id header.

🟡 POST - Resource Creation

POST /register

User Account Creation

Validation: Name must contain English letters only. Password must be min 6 chars.

Request Body:

{
  "name": "Neomi Levi",
  "email": "neomi@example.com",
  "password": "password123",
  "password_confirmation": "password123"
}

POST /movies API KEY

Add New Movie to DB

Requires full movie data. Returns 409 Conflict if the title already exists.

POST /payments/checkout

Secure Checkout (Payment)

Full order flow with integrated payment simulation. Pricing is calculated automatically:

  • Back Seats (Rows A-B): ₪35.00
  • Standard Seats (Rows C-D): ₪50.00
  • Premium Seats (Row E): ₪65.00

Required JSON:

{
  "userId": 1, "movieId": 1, "seats": ["A1", "E1"],
  "cardHolder": "Alice Cohen", "cardNumber": "1234567812345678",
  "expiry": "12/28", "cvv": "123", "date": "2026-10-10", "time": "20:00"
}

🔵 Management & Maintenance

PUT /movies/:id API KEY

Full Movie Update

Update all fields of an existing movie. Requires valid ID and Admin Key.

DELETE /movies/:id API KEY

Hard Delete Movie

Permanently removes a movie from the database.

DELETE /test/reset API KEY

QA System Reset

Deletes all current orders and resets movies and users to the initial factory state.
